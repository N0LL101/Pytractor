
# Pytractor Tool
It is a tool for collecting subdomains and endpoints.


## Install

To deploy this project run

```bash
 1- git clone https://github.com/N0LL101/Pytractor.git
  
 2- cd Pytractor
  
 3- python3 pytractor.py
```
